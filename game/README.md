

# GAME 

This project is a simple game for beginners.


## Technologies

- Node.js
- readline-sync
  
## Setup

1. Clone the project:

    ```bash
    git clone https://github.com/Ulyk04/backend_projects.git
    cd backend_projects/game
    ```

2. Install the necessary dependencies:

    ```bash
    npm install
    ```


3. Start the server:

    ```bash
    npm start
    ```
