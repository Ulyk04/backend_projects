

# API-users

This project is a simple API program for beginners.


## Technologies

- Node.js
- body-parser
- express
- pg
- bcrypt
  
## Setup

1. Clone the project:

    ```bash
    git clone https://github.com/Ulyk04/backend_projects.git
    cd backend_projects/API-users
    ```

2. Install the necessary dependencies:

    ```bash
    npm install
    ```


3. Start the server:

    ```bash
    npm start
    ```
